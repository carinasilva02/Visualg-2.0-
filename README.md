# Visualg-2.0-
Primeiro exercício de Algoritmo

Este repositório contém meu primeiro exercício de lógica de programação desenvolvido na disciplina Algoritmo II.

O objetivo foi praticar conceitos básicos utilizando Visualg, como:

- Declaração de variáveis
- Entrada de dados com "leia"
- Saída de dados com "escreval"

O algoritmo solicita o nome e sobrenome do usuário e exibe o nome completo na tela.

Exercício desenvolvido para aprendizado em programação.
